艦これ IME辞書登録用ファイル

この zip には以下のファイルが含まれています。

  readme.txt     このファイル
  ship1.txt      艦船名（人名扱い）
  ship2.txt      艦船名（固有名詞扱い）
  shiptype.txt   艦種
  weapon.txt     装備・装備種
  misc.txt       その他の艦これ関連用語

ship1.txt と ship2.txt は収録されている単語は同一です。改造済艦船名（「吹雪改」など）は含まれません。

ship1/2.txt では、システム辞書に含まれているかどうかに関わらず、全ての艦船名を収録しています。
その他のファイルでは、手元のシステム辞書に含まれている単語は収録していないことがあります。

これらのファイルは MS-IME 形式で、文字コードはシフトJIS (CP932) です。
最近のメジャーな IME は他のメジャーな IME の形式を読み込むことができます。
ATOK と Google IME では読み込めたという報告があります。

ソースは GitHub https://github.com/umezawatakeshi/kancolle-dic で公開しています。
リポジトリでは適宜修正が行われているため、最新状態が必要な場合はそちらを参照してください。

このリポジトリ(Cluele/kancolle-dic)はフォーク元である梅澤氏の艦これIME辞書登録用ファイルに未登録の艦娘・地名・武器・艦種等に関する用語を追記することを目的としています。

この zip には以下のファイルが含まれています。

  readme.txt     このファイル
  ship1.txt      艦船名（人名扱い）
  ship2.txt      艦船名（固有名詞扱い）
  shiptype.txt   艦種
  weapon.txt     装備・装備種
  misc.txt       その他の艦これ関連用語

ソースは Github https://github.com/Cluele/kancolle-dic/ で公開しています。
リポジトリでは新艦娘実装や新イベント後に更新を行っていく予定です。必要な場合はアップデートしてください。


以下は開発者向けのメモです。IMEから辞書登録をするだけの場合は下記を行う必要はありません。

makefileを使用してship2.txtを生成する場合、Perlとmakeの実行環境を用意してmakeコマンドを実行する必要があります。
また、Release用zipファイルを生成するにはzipコマンドを使用できる必要があります。
Windowsユーザーの場合はGnuWin32などのGNU Toolsをインストールして環境変数パスを通し環境を構築してください。
そしてPerlの実行環境を同Windowsマシン上に用意してください。
